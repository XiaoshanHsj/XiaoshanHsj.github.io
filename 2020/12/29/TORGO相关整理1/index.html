Automatic Speech Recognition with Deep Neural Networks for Impaired Speech

Database

In total, 5586 utterances for speakers with dysanthria and 10931 utterances for control speakers.

System’s Architecture

Feature Extraction

features: 13 MFCCs; 40 filterbank features

windows: 25ms

shift: 10ms for control speakers; 15ms for dysanthric speakers

SI: LDA (40 dimensions) + MLLT

SD: fMLLR; 100-dimensional iVectors

Acoustic Modeling

Classical Architectures

1800 HMM states and a total of 9000 Gaussians

Architecture description
MONO Monophone model with MFCC features
TRI Basic triphone model with features MFCC+Δ+ ΔΔ
TRI-SI Triphone model applied MFCC+LDA+MLLT
TRI-SD Triphone model added MFCC+LDA+MLLT+fMLLR
TRI-SDdis TRI-SD with a discriminative MMI and a feature-space MMI training. (learning rate = 0.001)

subspace Gaussian Mixture Models with 8000 states and 19000 substates

Architecture description
sGMM Subspace GMM on top of SD features MFCC+LDA+MLLT+fMLLR
sGMM2 Subspace GMM with fMLLR

Neural Network Architectures

alignments is obtained with MFCC+LDA+MLLT+fMLLR features

Architecture description
DNNCE DNN with cross-entropy. 6 hidden layers, 1024 neurons and 1800 output units. initialised with stacked RBMs
DNNsMBR Departing from DNNCE, 6 iterations of state-level minimum Bayes risk (sMBR) are applied
CNNba CNN with convolution along the frequency axis. 2 convolutional layers and a learning rate of 0.008
CNNsMBR CNNba + DNNsMBR
TDNN high-resolution MFCC features. 3 hidden layers with p-norm input dimension of 2000 and output dimension of 250. learning rate from 0.01 to 0.007
TDNNiV add 100-dim iVectors to the 40-dim high-resolution MFCC input features
LSTM 3 hidden layers with 1024 neurons. learning rate from 0.0012 to 0.00036. momentum 0.5
LSTMiV add 100-dim iVectors to the 40-dim high-resolution MFCC input features

LM and Pronunciation Lexicon

3-gram LM with interpolated Kneser-Ney discounting on the training data transcripts

for the lexicon, we choose the Carnegie Mellon University Pronouncing Dictionary for North American English

Results

use 14 speakers for training and test the systems on the 15th.

For cross-validation in neural networks training, use the data coming from a speaker with mild dysarthria speaker.

System’s Architecture

Acoustic Modeling

GMM-HMM Acoustic Modeling

Frame Rate and Word Position Dependency of Phones

use frame shift of 15ms for dysarthric speakers and position independent phone sets

Number of Context Dependent States

The total number of Gaussians was fixed at 9000.

Dimension of LDA Features

choose the dimensionality of LDA features as 30.

Speaker Normalized FMLLR Features

FMLLR feature are also 30-dimensional.

DNN-HMM Acoustic Modeling

mini-batch size of 256 frames; learning rate of 0.008; FMLLR features were stacked over a context window of 11 frames (±5)

Number of Hidden Layers and Nodes

used 4 hidden layers with sigmoid activation functions and 1024 neurons in each layer for all DNNs.

Dropout for DNN Generalization

apply a dropout factor of 0.2 for the first four DNN training epochs

Dysarthric Speech Recognition with Lattice-free MMI

It shows that time-delay neural network (TDNN) acoustic models trained with the LF-MMI objective

Database

System’s Architectures

leave-one-out cross-validation;

split the evaluation of isolated and multi-word utterances by treating the two tasks separately

HMM/GMM

features与第一篇相同; model phones independent of their position in words

HMM/DNN

hybird HMM/DNN models are based on the well-tuned Kaldi recipes for the 5-hour subset of the Librispeech corpus.

the main system is a 13-layer factorised TDNN model trained with the sequence-discriminative LF-MMI objective function.

speed perturbed version at 0.9 and 1.1 times the original speed.

LM

isolated word: unigram model; sentences: bigram LM

trained the LMs on the data of all speakers

Results

LF-MMI models make much fewer insertion errors

approximate speaking rate information by computing mean phoneme durations

speed perturbation

To test the effects of speed perturbation, train LF-MMI models on the original data only, but increasing the number of epochs by a factor of 3 to compensate for the lower amount of training data.

Frame shift

10ms for the entire data does not reduce performance on dysarthric speech. This is useful because the constant frame shift does not require prior knowledge about the speaker.

Dysarthric Speech Recognition using Convolutional Recurrent Neural Networks

database

only the following label of classes out of 530 classes have between 30 to 50 clips per class

except the words number 1,3 and 14, in total with 16 words.

Divide these 16 data samples as 80% for training subset, 20% for testing subset.

Features

MFCC; samples rate as 16000, clip duration as 1000 milliseconds, duration of frequency analysis window as 30ms, window stride as 10ms

System’s Architectures

cross entropy mean, softmax of a cross entropy with logits and the ADAM optimizer

CNN

first layer: filter with standard deviation 0.01 and size as 20(filter height) x 8(filter width) x 64(filter number) x 1 (stride), bias = 0, ReLU activation function, dropout function; (max)pooling layer with filter and stride as 1 x 2 x 2 x 1

second layer: filter with standard deviation 0.01 and size as 10 x 4 x 64 x 64, bias = 0. Finally, flatten the shape to be in 1-dim

third layer: fully connected layer

CRNN

first CNN block: convolution layer(MFCC,filter size as 10 x 4 x 48 x 1, valid padding, 2 x 2 stride) + ReLU layer + dropout layer

second block: unidirectional RNN block(two RNN layers include 60 neurons, GRU) + ReLU layer + dropout layer
