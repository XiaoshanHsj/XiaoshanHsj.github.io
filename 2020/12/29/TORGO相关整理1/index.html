Neural Network Architectures

вҖӢ alignments is obtained with MFCC+LDA+MLLT+fMLLR features

Architecture description
DNNCE DNN with cross-entropy. 6 hidden layers, 1024 neurons and 1800 output units. initialised with stacked RBMs
DNNsMBR Departing from DNNCE, 6 iterations of state-level minimum Bayes risk (sMBR) are applied

TDNN high-resolution MFCC features. 3 hidden layers with p-norm input dimension of 2000 and output dimension of 250. learning rate from 0.01 to 0.007
TDNNiV add 100-dim iVectors to the 40-dim high-resolution MFCC input features
LSTM 3 hidden layers with 1024 neurons. learning rate from 0.0012 to 0.00036. momentum 0.5
LSTMiV add 100-dim iVectors to the 40-dim high-resolution MFCC input features
