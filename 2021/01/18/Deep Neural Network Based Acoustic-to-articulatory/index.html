Deep Neural Network Based Acoustic-to-articulatory Inversion Using Phone Sequence Information

发表于 2021-01-18 更新于 2021-01-19 分类于 Acoustic-to-articulatory
Deep Neural Network Based Acoustic-to-articulatory Inversion Using Phone Sequence Information论文学习

Abstract

基于神经网络的acoustic-to-articulatory inversion方法表现是state-of-the-art的，但是有个问题就是转换的时候缺少音素序列信息。为了解决这个issue，这篇paper提出了一个将音素分类和inversion串联的架构来提高articulatory动作的生成。

在一个普通话转换任务上，提出的方法超过了DNN和RNN的baseline，这些架构要么使用没有因素序列的信息，混合密度参数的输出层，在输入层使用额外音素特征，或者多任务学习额外的单音素输出层目标标签，都使用EMA均方根误差和相关系数衡量。

使用这个结构中提取出来的瓶颈层作为辅助特征，基于generalized variable parameter HMMs的转换系统可以得到更大的提升。

Introduction

movements of articulators可以作为语音声学代表的可视化替代品。准确的内外articulatory movements是通过electromagnetic articulography (EMA)记录的。但是由于记录大量的高质量articulatory movements的代价太高了，因此从声学数据转换到articulatory movements的统计方法经常被使用。这些转换方法可以被分为两大类。

第一类是生成模型，使用HMMs来对acoustic和articulatory数据进行联合建模，比如一些有效的模型比如multiple regression HMM和generalized variable parameter HMMs。这些方法的一个问题就是需要句子的音素序列信息。一般来说，得到转录标注是不简单的，同时使用ASR的话会引入错误导致转换效果有明显的下降。

第二种是广义的非线性回归技术，以ANNs为代表。早期研究使用传统的浅层MLPs。这些模型最开始用来直接从acoustic特征映射到articulatory movements，之后用MDNs或者RNNs来生成articulatory trajectory 密度分布参数。这些技术利用了神经网络固有的强大泛化性能和序列建模能力。训练阶段不用使用phone或者viseme信息。相反，它们是通过隐藏层的呈现，以一种无监督的方式隐性学习的。因此不同音素或者viseme单元的相邻acoustic或者articulatory帧的区别是不会被学习到的。这被发现在转换过程中会产生噪音articulatory movement trajectories。

为了解决这个问题，叠加的DNN和RNN转换方法被提出来。一个使用声学特征输入的DNN或者RNN底层音素分类用来产生瓶颈特征。这些内容使用传统的声学前端进行增强然后喂到一个顶层的DNN或者RNN网络articulatory转换器中。

ANN based articulatory inversion

![image-20210118230508000](H:\github page\Hexo\source_posts\Deep Neural Network Based Acoustic-to-articulatory\image-20210118230508000.png)

这两个模型的输入包括在每个时间实例上构建的声学特征的上下文窗口，隐藏层可以是feedforward层或者recurrent层。在MLP和DNN的a2a转换网络中，Sigmoid或者线性激活函数用在输出层。静止的articulatory特征用来当作有监督的标签。相反的，一个特别设计的，用于预测GMM基准的articulatory trajectory参数的输出层用在MDNs或者深层MDNs中。通常，HMM方法中GMM层中会使用一个单高斯组件。在转换的时候，先计算每一帧的高斯组件参数，之后用最大似然参数生成(MLPG)算法来生成静止的articulatory feature。

Hierarchical NN based articulatory inversion

这个stacked, hierarchical ANN architecture包含了一个底层的分类子网络和一个顶层转换子网络。底层音素分类器是一个DNN或者RNN，它的输入时声学特征，输出是单音素标签，这个分类器可以用来生成瓶颈特征。瓶颈特征可以代表音素序列信息的潜在表示；瓶颈特征加上传统的声学特征，然后送到顶层的articulatory转换DNN或者RNN自网络中。如图3所示。

![image-20210119165543810](H:\github page\Hexo\source_posts\Deep Neural Network Based Acoustic-to-articulatory\image-20210119165543810.png)

构建这样一个NN系统主要分成两个阶段。第一个阶段，底层网络使用传统的交叉熵准则。在第二个阶段，单音素分类器产生的瓶颈层特征被加到acoustic特征上，组成上下文的tandem特征来训练这个转换网络。但是这个阶段有两个issue需要解决。

首先，为了保证训练过程中稳定和快速的收敛，最顶层的子网络使用一个传统的DNN/RNN转换器模型进行初始化，传统的转换器模型是没有瓶颈特征的，因此瓶颈特征的权重的初始化是随机的。第二个就是只有输入层与瓶颈特征有关的权重需要全部更新，而剩余的权重只需要微调，这样的话要对这两个部分设置不同的学习率才行。为了让转换子网络处理瓶颈特征更加高效，输入层连接瓶颈特征的权重乘上一个因子α=2。

Hierarchical inversion bottleneck feature for tandem GVP-HMM based inversion

当测试句子的音素序列是知道的，hierarchical inversion NN 系统也能当作辅助特征提取器，辅助特征用于generalized variable parameter HMMs (GVP-HMMs)可以得到更多的提升。这需要在顶层的转换子网络中添加一个瓶颈层，如图4

![image-20210119194041961](H:\github page\Hexo\source_posts\Deep Neural Network Based Acoustic-to-articulatory\image-20210119194041961.png)

对于帧t的N维的瓶颈特征向量表示为
$$
f_t^{BN}=[v_{t,1}, …, v_{t,j}, …,v_{t,N}]
$$
多项式的阶为P，一个P*N+1维度的Vandermonde向量构建如下
$$
v_t^{T}=[1, \hat f_{t,1}, …,\hat f_{t,p},…,\hat f_{t,P}]^{T}
$$
N维的第p个子向量定义为
$$
\hat f_{t,p} =[v_{t,1}^p,…,v_{t,j}^p,…,v_{t,N}^p]
$$
其中Vandermonde向量用来训练tandem GVP-HMMs。GVP-HMMs对articulatory和acoustic流的联合状态概率函数进行建模。在第t帧，对于一个给定的状态q，可以得到
$$
b_q(a_t,o_t)=\Nu(a_t;\mu_q^{(a)}(v_t),Σ_q^{(a)}(v_t))\Nu(o_t;\mu_q^{(o)},Σ_q^{(o)})
$$
其中at和ot是articulatory和acoustic特征。

GVP-HMMs提供了articulatory和acoustic复杂关系的灵活建模方式。高斯均值和方差的trajectory函数uq(·)，Σq(·)能够被给定的辅助特征的P阶多项式表示。当对角协方差被使用时，第i维的均值和方差的trajectory是
$$
\mu_{q,i}^{(a)}(v_t)=v_t·c^{(\mu_{q,i}^{(a)})} \
\sigma_{q,i,i}^{(a)}(v_t)=\hat \sigma _{q,i,i}^{a} \ v_t·c^{(\sigma_{q,i,i}^{(a)})}
$$
其中c(·)是P*N+1维的多项式系数向量，

![image-20210119214813556](H:\github page\Hexo\source_posts\Deep Neural Network Based Acoustic-to-articulatory\image-20210119214813556.png)是传统的HMM方差估计。系数向量可以用最大似然更新方案来估计。利用高效的贝叶斯模型复杂度控制技术，可以优化不同参数的多项式阶数P所代表的GVP-HMM模型底层结构。有了GVP-HMM状态参数trajectories，最终的articulatory movements就可以使用MLPG算法生成了。

Experiments

EMA data and experiment setup

普通话语音和相关的EMA数据用一个Carstens AG-501 EMA设备进行记录。忽略一些小的movements，按上唇、下唇、舌背、tongue dorsum、舌尖和下颌的x-和y-坐标，以及口腔右角（与左角对称）的z-坐标，依次选取每一帧的13维静态EMA特征向量，并通过将平均静态位置设置为零进一步归一化。39维的MFCC声学特征从语音中提取。利用均方根误差(RMSE)和相关性来评估预测的EMA数据。其中训练、验证和测试数据分别包括2950，50和50个句子，总共2.9小时。

articulatory转换DNNs包括5个512个神经元的前向层。作比较的RNN有3个每层512个神经元的前向层，2个128个单元的LSTM层和用于维度减少的递归投影层。一个11帧MFCC的上下文窗口只选择每隔一帧作为其输入，13个静态EMA特征作为目标。

分类DNN和RNN使用相同的声学特征作为输入，118个tonal单音素标签作为输出目标。分类子网络产生39维的特征，然后和声学特征串联后来训练articulatory转换DNN或者RNN。除此之外，音素输入特征和多任务学习转换ANNs也会被训练，MDN也是。最后每个系统表现最好的特征会被用来产生GVP-HMM的辅助特征。Kaldi用来训练所有的神经网络，HTK用来训练GVP-HMM。对于所有的结果，基于统计意义测试的paired t-test用RMSE reduction.
