Finite State Transducers

WFST: 8 tuple written as (Σ, Δ, Q, E, I, F, λ, ρ)， K is weights

  • Σ: Set of input labels
  • Δ: Set of output labels
  • Q: Finite set of states
  • E: Finite set of state transitions. E ⊆ Q × {Σ ∪ ε} × {Δ ∪ ε} × K × Q
  • I: Set of initial states
  • F: Set of Final states
  • λ: Initial weight function (λ: I ➡ K)
  • ρ: Final weight function (ρ: F ➡ K)

Given T: (Σ, Δ, Q, E, I, F, λ, ρ) and x ∈ Σ, y ∈ Δ

  • Deterministic: T has no two transitions leaving the same state sharing the same input label
  • Sequential: T has a unique initial state
  • Functional: T has a unique y for any x
  • Epsilon-Transitions: ∃ a state transition that can be made without any input (ε)
  • Stochastic WFST: For every state, the sum of all transitions leaving the state is exactly 1.

Semirings

Addition:

  • Associativity: x ⊕ (y ⊕ z) = (x ⊕ y) ⊕ z
  • Commutativity: x ⊕ y = y ⊕ x
  • Identity, 0’: x ⊕ 0’ = x
  • Inverse

Multiplication:

  • Associativity : x ⊗ (y ⊗ z) = (x ⊗ y) ⊗ z
  • Commutativity
  • Identity, 1’: x ⊗ 1’ = x
  • Inverse

Distributivity of Multiplication over Addition:

  • z ⊗ (x ⊕ y) = (z ⊗ x) ⊕ (z ⊗ y)
  • (x ⊕ y) ⊗ z = (x ⊗ z) ⊕ (y ⊗ z)

The algebraic structure above is known as a SEMIRING, Recall PROBABILITY.

Common Examples of Semirings

  • Probability: ([0,1], + , ×, 0, 1)
  • Strings: (Σ* ∪ S∞, Λ, ·, S∞, ε), where
    • Σ*: A finite set of strings
    • S∞: Infinite String
    • Λ: Lowest common prefix
    • · : Concatenation of 2 strings
  • Tropical: (R, min, +, ∞, 0) where
    • min: The minimum function
  • Log: (R, ⊕Log, +, ∞, 0), where
    • x ⊕Log y = -log(e^-x + e^-y)

Special Cases of Semirings

  • Commutative Semirings: if ⊗ is commutative as well

    • Tropical and Log are Commutative Semirings
    • The commutative property holds for classical addition

  • Idempotent Semiring: if x ⊕ x = x, for any x ∈ K

    • Tropical and String are Idempotent Semirings
    • Tropical: x ⊕ x = min(x, x) = x
    • String: x ⊕ x = x Λ x = x

    • String Semirings are 0-closed: 1’ = ε = ε Λ a = 1’ ⊕ a

  • Weakly Left Divisible Semiring: if any (x ⊕ y) ≠ 0‘, exist z ∈ K such that x = (x ⊕ y) ⊗ z

    • Probability Semiring is a “ZeroSum Free” Semiring

Basic Operations for Finite Automaton

WFST over semiring S = (K, ⊕, ⊗, 0’, 1’)

where λ(p[π]) and ρ(n[π]) are the weights of the initial and final states.

Given a WFST over a Tropical Semiring: (R, min, +, ∞, 0)

Consider π0 = (0, a, A, 1, 1), (1, c, B, 1, 3), x = {ac}, y = {AB}

T(x, y) = λ(p[π0]) ⊗ w[π0] ⊗ ρ(n[π0])

= λ(0) ⊗ (1 ⊗ 1) ⊗ ρ(3)

Basic Operations：

  • Kleene closure: Modifies the automaton so that the set of symbol sequences or transductions is sequentially repeated zero or more times
  • Union: Combines two automata in parallel
  • Concatenation: Combines two automata in series
  • Inverse: Exchanging input and output symbols
  • Reversal: Assigns to each pair of strings (x, y) what T assigns to its mirror images (x’, y’)
  • Projection: Converting a transducer into an acceptor by omitting input or output labels

WFST Epsilon Removal

To make these weighted transducers more efficient to use, it may be perferable to remove all ε-transitions

Description of Epsilon-Removal Algorithm

  • Compute the weighted ε-closure for each state in the FST

    weighted ε-closure:
    $$
    C(p)={(q,w):P(p, \epsilon ,q)≠\empty,w=\delta_\epsilon[p,q]}\ \forall p∈Q
    \
    Where\ \ \ \ \delta_\epsilon[p,q] = ⊕_{\pi∈P(p, \epsilon ,q)}w(\pi)
    $$

  • Remove all ε-transitions and ∀p ∈ Q and each (q, w)∈C(p), and the transition set of p is augmented to:

    • {(p, a, b, δε[p, q] ⊗ w, r) : (q, a, b, w, r) ∈ E, (a, b) ≠ (ε, ε)}
    • if exist (q, w) s.t. q ∈ F : w(p) ⬅ w(p) ⊕ (δε[p, q] ⊗ ρ(q))

WFST Weight Pushing

Many sequence recognition or transduction problems reduces to finding the minimal cost path; As the unpromising paths can be eliminatd in the early stage of the search, and therefore reducing the total search time.

e.g.

Description of Weight-Pushing Algorithm

  1. Compute potential of each state, given by
    $$
    V[q]=⊕_{\pi∈\Pi(q,F)} w(\pi) ⊗ ρ(n[\pi])
    $$
    Where ∏(q, F) is the set of paths originating from state q

  2. Re-weighting the transition weights, initial weights and final weights in the following way:
    $$
    \begin{aligned}
    & \forall e ∈ E\ s.t.\ V[p[e]]≠0,\ w[e]\leftarrow V[p[e]]^{-1} ⊗ w[e] ⊗ V[n[e]]\
    & \forall q ∈I, \ \lambda(q)\leftarrow \lambda(q)⊗V[q] \
    & \forall q ∈ F,\ ρ(q) \leftarrow V[q]^{-1} ⊗ ρ(q)
    \end{aligned}
    $$
