Learning Hidden Unit Contributions for Unsupervised Acoustic Model Adaptation

发表于 2021-01-22 更新于 2021-01-23 分类于 adaptation
Learning Hidden Unit Contributions for Unsupervised Acoustic Model Adaptation论文学习

Abstract

LHUC是一种使用少量无监督的适应数据，以依赖于说话人或环境的方式线性地重新组合隐藏单元的方法。

同时将LHUC扩展到说话人自适应训练（SAT）框架，从而得到一个适应性更强的DNN声学模型，可以以依赖说话人和不依赖说话人的方式工作，而不需要维护辅助的依赖于说话人的特征提取器，也不需要对DNN结构引入显著的依赖于说话人的变化。

通过对四个不同的语音识别基准（TED talks、Switchboard、AMI meetings和Aurora4）的一系列实验，包括270个测试演讲者，LHUC在其仅测试和SAT变体中都会导致一致的单词错误率减少，相对来说，减少幅度在5%到23%之间，这取决于任务以及训练和测试数据之间的不匹配程度。此外，我们还研究了每个说话人的适应数据量、无监督适应目标的质量、与其他适应技术的互补性、单次适应以及用序列区分方式训练DNNs的自适应。

Review of Neural Network Acoustic Adaptation

神经网络声学模型的自适应方法可以看作是在特征空间操作或者在模型空间操作，又或者是一个说话人、句子、环境相关的辅助特征被加入到标准的声学特征的混合方法。

估计特征空间转换的主要方法是constrained (feature-space) MLLR，也就是CMLLR或者fMLLR。fMLLR是一个为GMM声学模型而发展出来的自适应方法，其中输入声学特征的仿射(affine)转换会通过最大化模型生成适应数据（基于第一次对齐）的对数似然来估计。为了在DNN系统中使用fMLLR，就很有必要训练一个GMM系统，用来估计每个说话人的单一输入转换。转换了的特征向量用来训练说话人适应的DNN，对unseen说话人进行测试时估计另一组转换。该技术已被证明可以有效地降低几个不同数据集的WER，在混合和串联方法中都是如此。

The linear input network (LIN) 在输入特征和第一个隐藏层之间定义一个额外的说话人相关网络层，因此和fMLLR有相似的作用。之后发展出变量绑定的LIN，即每个输入帧被限制有相同的线性转换。LIN和tied-LIN被用在test-only的适应方案上；为了利用fMLLR变换，需要进行SAT训练，这通常可以更好地补偿声学空间的变化。
